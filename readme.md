# TradeShop

### _A simple, up-to-date, item-for-item trade chest shop plugin._

![Languages](https://badgen.net/badge/language/Java/red) ![Platform](https://badgen.net/badge/platform/Minecraft%201.21+/blue) ![License](https://badgen.net/badge/license/MIT/red)

# Usage
Download from Releases or Modrinth.
Install onto server.
Place sign on Chest/Shulker/Barrel
```
[Shop]
X ItemName
for
Y ItemName
```

*ItemName must be the name of the [Bukkit Material](https://jd.papermc.io/paper/1.21.1/org/bukkit/Material.html) (case/space doesn't matter)

Your username will appear on the chest in green if successful.

### Example
```
[Shop]
1 Iron Ingot
for
64 Wheat Seeds
```
